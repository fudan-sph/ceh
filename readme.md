# 儿童重要肿瘤与血液性疾病多模态数据分析平台 - 项目文件说明

## 项目概述
本项目是专注于儿童重要肿瘤与血液性疾病的多模态数据分析平台，依托“十四五”国家科技部重点研发计划项目（项目/课题编号：2022YFC2705000/2022YFC2705004），旨在整合多维度数据资源与分析工具，为疾病研究提供高效、安全的数据分析支持。平台相关文件涵盖项目介绍、使用指南、工具说明、帮助文档等核心内容，以下是各主要文件的详细说明。


## 核心文件说明

### `index.html`
- **功能**：平台首页，提供功能导航与核心入口
- **主要模块**：平台简介、快速功能入口（数据上传、工具选择、分析流程）、最新公告等


### `manual.html`
- **功能**：平台使用手册与环境配置指南
- **核心内容**：
  - R环境配置步骤：包括外观设置（主题/字体）、R包安装（`dplyr`/`ggplot2`等，需勾选依赖项）、代理设置（CRAN镜像配置）、环境验证方法
  - 数据分析流程指引：从数据上传到结果可视化的全流程操作说明


### `tools.html`
- **功能**：平台集成的数据分析工具清单及说明
- **分类及核心工具**：
  - 数据处理工具：Python库`Missingno`（缺失值可视化）、R包`tidyverse`（数据处理全家桶）、`data.table`（大型数据处理）等
  - 特征提取与深度学习：`CLAM`（弱监督学习框架）、`HistomicsTK`（病理特征计算）、`PyTorch Metric Learning`（表征学习）等
  - 空间分析与生物信息学：R包`spatialLIBD`（空间转录组可视化）、`scikit-bio`（生物信息统计）、`iNAP`（分子生态网络分析）等
  - 临床信息分析：R包`survival`（生存分析，含Kaplan-Meier曲线、Cox模型）
  - 多模态数据融合分析：标准化流程与可视化工具


### `help.html`
- **功能**：用户常见问题与平台规范说明
- **核心内容**：
  - 数据安全说明：数据仅授权用户访问，不对外共享
  - 隐私保护措施：权限分级管理、操作日志记录、传输存储安全保障
  - 平台使用常见问题解答


### `about.html`
- **功能**：介绍平台研发团队及项目背景
- **主要内容**：
  - 课题负责人信息：张蕴晖教授（复旦大学公共卫生学院，“上海亲子队列”负责人，全球前2%顶尖科学家）
  - 团队简介：隶属于复旦大学公共卫生学院环境卫生教研室，专注于环境与儿童健康、儿童肿瘤及血液性疾病研究
  - 研究方向：环境内分泌干扰物的生殖发育毒性研究、基于专病队列的儿童疾病遗传机制研究等

## 技术架构说明
- **前端技术**：基于HTML、Tailwind CSS、Font Awesome构建，支持响应式布局
- **工具集成方式**：通过标准化接口整合R/Python生态工具，支持本地化部署与浏览器端交互
- **数据安全框架**：基于权限分级的访问控制体系，符合医疗数据隐私保护规范
