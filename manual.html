注册与登录

本平台依托 RStudio Server 搭建，请联系平台管理员获取账号注册权限。

  • • 点击主页 “立即使用” 访问登录界面。
  • • 输入管理员分配的“用户名”和“初始密码”。
  • • 安全提醒： 首次登录后，请立即前往个人设置修改初始密码。

初始化配置

  • • 1. 工作目录 在右侧【Files】区域新建专用目录（如“R_Workspace”）并设为默认，控制台输入 getwd() 验证。
  • • 2. 界面个性化 通过【Tools】→【Global Options】→【Appearance】调整主题和字体大小。
  • • 3. R包配置 在【Packages】点击【Install】批量安装 dplyr, ggplot2 等常用包。务必勾选“Install dependencies”。
  • • 4. 代理设置 如遇网络问题，可在【Tools】→【Global Options】→【Packages】配置CRAN镜像。
  • • 5. 环境验证 通过【File】→【New File】→【R Script】新建脚本，输入测试代码运行，控制台正常显示结果即配置完成。

数据上传与导入

流程操作（以CSV为例）：

  • • 1. 数据上传：点击右侧【Files】区域的 Upload 按钮。
  • • 2. 数据导入：加载rio包，使用代码 import("文件名.csv") 运行导入。
  • • 3. 验证结果：在【Environment】区域查看对象是否成功生成。

运行分析查看

数据导入成功后，可开展基础数据分析与结果展示：

  • • 1. 数据清洗：使用dplyr等包进行数据查看、清洗、筛选排序等操作，结果实时显示在控制台。
  • • 2. 数据可视化： 通过ggplot2包绘制可视化图形展示结果。图形自动显示在右侧Plots区域，支持保存导出。
  • • 3. 结果保存： 使用write.csv()等函数将处理后的数据导出为CSV等格式文件，保存在工作目录中。
  • • 4. 报告生成： 若需生成正式报告，可新建RMarkdown文档整合代码、分析过程与结果，最终导出为PDF等格式。
  • • 5. 安全退出： 完成操作后，点击页面右上角的用户名，在下拉菜单中选择【Sign Out】，即可安全退出平台。
